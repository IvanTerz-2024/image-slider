# Слайдер изображений

Решение кейс-задачи № 3 по учебной практике.

Проект содержит:
- интерфейс на HTML;
- оформление на CSS;
- переключение изображений на ванильном JavaScript;
- кнопки «Назад» и «Вперёд»;
- зацикливание слайдера;
- счётчик текущего изображения.

## Запуск

Откройте файл `index.html` в браузере.
